# Tickedify Changes - December 27, 2025

## Summary
Major improvements to the Daily Planning screen with new edit functionality, UI enhancements, and various bug fixes. Version updated from 1.0.200 to 1.1.15.

---

## Features

### Edit Icons in Daily Planning
- Added edit icons (✏️) to both the Actions sidebar and Calendar view
- Click to open task detail popup directly without leaving your planning workflow
- Enables quick task editing while planning your day

### Click Outside to Close
- Planning popup now closes when clicking outside of it
- Provides a more intuitive and natural interaction pattern

### Search Loading Indicator
- Added visual loading indicator to the search screen
- Spinner appears after 400ms delay to avoid flashing on fast searches
- Enlarged spinner (40px) centered for better visibility

---

## Improvements

### Daily Planning UI
- **Date format**: Changed from MM/dd/yyyy (US) to dd/MM/yyyy (European format) in the action list
- **Calendar checkboxes**: Enlarged by 50% for easier task completion
- **Icon spacing**: Reduced spacing between icons for a cleaner, more compact layout

---

## Bug Fixes

### Calendar Hours Reset
- Fixed issue where daily planner calendar reset to 08:00-18:00 instead of 05:00-22:00 when completing a task
- Caused by inconsistent default values in the task completion callback

### Task Name Display
- Fixed calendar not showing actual task name after editing a task
- Calendar view now properly refreshes with updated task text

### Edit Icons Functionality
- Fixed multiple issues with the new edit icons in Daily Planning
- Ensured proper event handling and popup positioning

---

## Other Changes

### Go High Level Integration
- Temporarily disabled Go High Level integration
